# Soul

## Core promise

VaultFront is a browser-based real-time strategy game where **precision timing and convoy
risk-taking beat expansion alone.** Every session is a live puzzle of when to capture,
when to protect, and when to strike — with enough comeback mechanics that players on the
back foot always have a path to victory.

## Non-negotiables

- **Depth without friction:** All six vault mechanics (vault sites, convoys, escort,
  beacon/jam, surge, execution chain) must be learnable within 2 matches and masterable
  over 20. Never add complexity that doesn't reward understanding.
- **Every match has a turning point:** The comeback surge and execution chain exist to
  prevent blowouts. If a session ever feels mathematically decided before it ends,
  something is wrong.
- **VaultSpark quality bar:** Animation feedback, sound cues, and HUD clarity must match
  or exceed what players expect from a polished web game. No rough edges ship.

## Audience

Lapsed strategy game players (18–35) who want depth and fast sessions (~20 min) without
a 40-hour learning curve. They value cleverness over grind.

## Anti-goals

- must never drift into: A reskin of OpenFrontIO with different names — VaultFront must
  be identifiable by its mechanics within 60 seconds of play.
- cheap imitation to avoid: Copying RTS tropes (resource nodes, build orders) without
  the vault twist that makes each fight about risk-scored logistics.
- tonal failure mode: Generic mid-core strategy game. VaultFront earns its name — every
  match should feel like a heist with a leaderboard.
